# prawl Documentation

Documentation site for prawl - Browser automation for AI agents

**Live URL**: https://prawl.dirtroad.dev

## Quick Start

```bash
cd docs
bun install
bun run dev      # Start dev server
bun run build    # Build for production
```

## What's Inside

- 📚 Complete command reference
- 🚀 Quick start tutorials  
- 🤖 Agent integration guides
- 📱 Mobile testing examples
- 🔧 API documentation
- 💡 Real-world workflows

## Deployment

Deployed automatically via GitHub Actions to `prawl.dirtroad.dev`
