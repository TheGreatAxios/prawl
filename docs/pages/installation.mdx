# Installation

## macOS

### Homebrew (Recommended)

```bash
brew tap thegreataxios/peep
brew install peep
```

### One-Liner Install

```bash
curl -fsSL https://raw.githubusercontent.com/thegreataxios/peep/main/scripts/install.sh | bash
```

### From Source

Requires [Bun](https://bun.sh) 1.3.12+:

```bash
git clone https://github.com/thegreataxios/peep.git
cd peep
bun install
bun run build
bun run compile  # Creates ./peep executable
sudo mv peep /usr/local/bin/
```

## Linux

### From Source

```bash
git clone https://github.com/thegreataxios/peep.git
cd peep
bun install
bun run build
bun run compile
sudo mv peep /usr/local/bin/
```

**Note**: Linux requires Chrome backend (WebKit not available). Chrome must be installed.

## Windows

Windows support is experimental. Use WSL2 with Linux instructions.

## Verify Installation

```bash
peep --help
peep --version
```

## Prerequisites

### macOS
- macOS 11.0+ (Big Sur or later)
- No additional requirements for WebKit backend
- Chrome optional for advanced features

### Linux
- Chrome or Chromium must be installed
- Required for all operations (no native WebKit)

### All Platforms
- [Bun](https://bun.sh) 1.3.12+ (for development/building from source)
